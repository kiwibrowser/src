The Hello World Stdio example is the simplest one in the SDK. It uses the ppapi_main library which creates a Module and an Instance, using default values to simplify setup and communication with the PPAPI system. In addition, it uses the nacl_io library to remap IO to the Pepper API. This simplifies IO by providing a standard blocking API and allowing STDERR to go to the JavaScript console by default.

In main, we write to both STDOUT and STDERR, printing a hello world message.
